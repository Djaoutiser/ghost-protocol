👻 Ghost Protocol SIEM

A lightweight Security Information and Event Management (SIEM) dashboard built with HTML, CSS, and JavaScript.

This project simulates a real-time cybersecurity monitoring interface where security events are displayed dynamically and threat metrics are updated instantly.

✨ Features
Live security event feed
Dynamic threat level updates
AI confidence score simulation
Event counter
Responsive dashboard layout
Modern dark cybersecurity UI
Random attack event simulation

🛠️ Technologies Used
HTML5
CSS3
Vanilla JavaScript

🚀 How to Run
Clone the repository

git clone https://github.com/yourusername/ghost-protocol-siem.git
Open the project folder
Launch index.html in your browser
No installation or dependencies are required
